# Summary

This is an examination for the Front-End Developer role at Tri Petch IT Solutions company.

## Getting Started

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/tomhannarong/Tri-Petch-IT-Solutions.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

### Usage

1. Start the server:

   ```bash
   npm run dev
   ```
